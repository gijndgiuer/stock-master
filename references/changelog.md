## v3.4.0 (2026-01-22)
- **K线形态识别**: 识别经典K线形态
  - 单K线: 十字星、锤子线、上吊线、射击之星、倒锤子
  - 双K线: 看涨吞没、看跌吞没
  - 三K线: 早晨之星、黄昏之星、三只白兵、三只乌鸦
- **趋势形态识别**: 识别经典图表形态
  - 反转形态: 双底(W底)、双顶(M头)、头肩顶、头肩底
  - 整理形态: 上升三角形、下降三角形、对称三角形
- **形态小白解读**: 用通俗语言解释每种形态的含义和操作建议
- **评分系统升级**: 形态信号纳入评分，强形态±3分，中等形态±2分，弱形态±1分
- 新增 `identify_candlestick_patterns()` 函数
- 新增 `identify_chart_patterns()` 函数
- 新增 `analyze_patterns()` 综合形态分析函数
- 新增 `explain_patterns_simple()` 形态解读函数

## v3.3.0 (2026-01-22)
- **KDJ 随机指标**: 国内股民最常用的短线指标，支持金叉/死叉/超买超卖判断
- **MACD/RSI 背离检测**: 自动检测顶背离和底背离，强烈反转信号
- **增强版支撑阻力位**: 斐波那契回撤位 + 扩展位 + 整数关口
- **OBV 量能指标**: 能量潮分析，检测资金流向
- **威廉指标 Williams %R**: 短期超买超卖判断
- **乖离率 BIAS**: 股价偏离均线程度分析
- **评分系统优化**: 调整阈值，背离信号权重最高 (±4分)
- **止损止盈优化**: 结合支撑阻力位优化价格建议
- 新增多个小白解读函数

## v3.2.0 (2026-01-21)
- **港股本地计算**: 自动检测港股，使用 Yahoo Finance 本地计算所有指标
- **优雅降级**: Alpha Vantage API 失败时自动回退到本地计算
- **ATR 动态止损**: 根据股票波动性自动调整止损距离
- **成交量分析**: 量价配合分析（放量/缩量 + 涨跌）
- **均线系统**: MA5/10/20/60 + 多空排列判断
- **仓位建议**: 根据信号强度建议仓位百分比（高信号30%/中20%/低10%）
- **风险收益比**: 自动计算并显示（默认目标 2.5:1）
- 新增 `scripts/indicators.py` 本地指标计算模块
